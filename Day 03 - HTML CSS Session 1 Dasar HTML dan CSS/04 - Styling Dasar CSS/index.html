Tentang Festival

Festival Kesenian ini adalah perayaan seni, budaya, dan tradisi yang akan berlangsung selama 3 hari penuh dengan berbagai acara menarik.

Jadwal Acara

  • Hari 1: Pembukaan dan Pameran Seni
  • Hari 2: Pertunjukan Tari dan Musik Tradisional
  • Hari 3: Penutupan dan Festival Makanan
